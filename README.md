# MY-To-Do-App
MY To-Do App is a desktop task management application built using Python, Tkinter, and SQLite. It allows users to add, view, edit, delete, search, and mark tasks as completed. All tasks are stored in a SQLite database, providing persistent storage and an easy-to-use graphical user interface.

Features:
- Add Tasks
- View Tasks
- Edit Tasks
- Delete Tasks
- Search Tasks
- Mark Tasks as Done
- SQLite Database Integration
- Tkinter GUI
